Markdown render from Ruby China
============================

## 如何使用
### 添加到Gemfile
```ruby
gem 'markdown-ruby-china'
```

### 调用
```ruby
MarkdownTopicConverter.format(content)
```

## 新功能
1.	代码高亮 自动识别程序语言，文件名，别名等多种格式。
2.	支持表格

```
| header 1 | header 2 |
| -------- | -------- |
| cell 1   | cell 2   |
| cell 3   | cell 4   |
```
